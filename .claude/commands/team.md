Erstelle ein Agent Team basierend auf der gegebenen Aufgabe.

Standard-Team für Gold Foundry:
1. **Backend Engineer** — src/lib/, src/app/api/. Import-Konventionen, Token-Effizienz.
2. **Frontend Engineer** — src/app/**/page.tsx, src/components/. Design System, Naming.
3. **QA Tester** — Prüft alles. 15-Punkte Compliance. Letztes Gate.

Erweitere das Team bei Bedarf um:
- **SEO Engineer** — Wenn SEO-Seiten gebaut werden
- **Trading Specialist** — Wenn Trading-Logik involviert ist
- **DevOps Engineer** — Wenn Build/Deploy Probleme gelöst werden

Reihenfolge: Backend zuerst → Frontend + SEO parallel → QA → Deploy
