name: orchestrate
description: Orchestrate multiple sub-agents to work on a complex task in parallel. Dispatches backend, frontend, SEO, trading, QA, and devops engineers as needed.
---
Run the orchestrator agent. Analyze the task, break it into sub-tasks, 
identify dependencies, and dispatch specialized sub-agents in parallel worktrees.

After all sub-agents finish, merge their branches and run QA.
