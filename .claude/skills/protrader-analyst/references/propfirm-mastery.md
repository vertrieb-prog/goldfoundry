## Prop-Firm Consistency Rule — Der stille Killer

Die meisten Trader scheitern nicht am Profit Target. Sie scheitern an der **Consistency Rule**.

```
CONSISTENCY RULE:
Kein einzelner Trading-Tag darf mehr als 30-40% des Gesamtgewinns ausmachen.

BEISPIEL:
→ Gesamtgewinn: $8,000
→ Bester Tag: $4,500 (56% des Gesamtgewinns)
→ ERGEBNIS: ABGELEHNT trotz 8% Profit.
```

### Wie man die Consistency Rule besteht

```
STRATEGIE: "Spread the Wealth"

1. TÄGLICHES PROFIT-CAP: Max $800 pro Tag ($100k Account)
   → Ab $800: STOP. Auch wenn du $2,000 hättest machen können.
   → Lieber 10 Tage à $800 als 2 Tage à $4,000

2. PROFIT-BANKING:
   → Ab +$500 Tagesprofit: Stops auf Breakeven
   → Ab +$800: Alle Positionen schließen, Tag beendet
   → NICHT "noch einen Trade" machen

3. VERLUST-MANAGEMENT:
   → Verlust-Tag limitieren auf max $500
   → "2 Losses and Done" Regel
   → Nächster Tag: halbe Size → verhindert Revenge-Trading

4. TRACKING:
   → Täglich notieren: P&L, Anzahl Trades, bester Trade
   → Wenn bester Tag > 25% des bisherigen Gesamtgewinns: BREMSEN
   → FORGE AI berechnet Consistency-Score automatisch
```

### Consistency Score Berechnung

```
Score = 1 - (Best Day P&L / Total P&L)

Optimal: Score > 0.70 (bester Tag < 30% des Totals)
Akzeptabel: Score 0.60-0.70
Gefährdet: Score < 0.60 → SOFORT Tagesprofit-Cap senken
```

## Position Sizing Bibel

```
REGEL 1: NIEMALS feste Lot Size
→ IMMER vom Stop berechnen: Lots = Risk$ / (StopPips × PipValue)

REGEL 2: NIEMALS mehr als 1% pro Trade in der Challenge
→ $100k Account = max $1,000 Risk pro Trade

REGEL 3: NIEMALS mehr als 2% kombiniertes Risiko offen
→ 2 Trades à 0.5% = 1.0% combined → OK
→ 3 Trades à 0.8% = 2.4% combined → ZU VIEL
```

## Funded Account — Langfristig überleben

Challenge bestanden. Jetzt beginnt das echte Spiel. 80% der Funded Trader verlieren ihren Account in den ersten 3 Monaten.

```

```

### Das "Payout Cycle" System

```
Monat 1: Konservativ. Target 2%. Beweise Konsistenz.
Monat 2: Leicht aggressiver. Target 3%. Consistency Score >0.70.
Monat 3: Normal. Target 3-4%. Erste Payout-Anfrage.
Monat 4+: Scaling beantragen. Kapital verdoppeln.

GOLDEN RULE: Nie mehr als 50% des Monatsgewinns im Account lassen.
→ Verdient $3,000? → $2,100 auszahlen (70% Split), $900 bleibt
→ Dadurch wächst der Buffer gegen DD-Reset nicht unkontrolliert
→ UND du realisierst echten Cashflow
```

### Drawdown Recovery Playbook (Funded)

```
DD < 2%: Normal. Weitertraden.
DD 2-4%: Size um 30% reduzieren. Nur beste Setups.
DD 4-6%: Size um 60% reduzieren. Nur Gold London. 1 Trade/Tag max.
DD 6-8%: PAUSE. 2-3 Tage nicht traden. Mindset reset.
DD 8-9%: Micro-Lots only. 0.1% Risk. Nur 1 Trade. Nur wenn PERFEKTES Setup.
DD 9%+:  STOP. Konto schützen. Warte auf Monats-Reset wenn Firm das erlaubt.

RECOVERY MATH:
→ Bei 5% DD musst du 5.26% Gewinn machen um auf 0 zu kommen
→ Bei 8% DD musst du 8.70% Gewinn machen
→ Je tiefer der DD, desto schwieriger Recovery UND du tradest mit weniger Size
→ DESHALB: DD VERMEIDEN > DD RECOVERN
```
