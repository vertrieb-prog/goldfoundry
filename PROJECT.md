# GoldFoundry Git (Ältere Version)

## Status
Aktiv (Git-Repo Version)

## Zweck
MetaTrader AI Analytics Portal. Trading-Dashboard für Prop-Firms mit MetaApi Integration. Ältere Version des GoldFoundry Projekts mit Git-History.

## Tech Stack
- Framework: Next.js 14.2.0 (App Router)
- React: 18.3
- DB: Supabase (PostgreSQL)
- Trading: MetaApi.cloud
- 3D: Three.js + react-three/fiber
- Charts: Lightweight Charts (WebGL)
- Payments: Stripe
- Telegram: Bot Integration
- CSS: Tailwind CSS v3

## Unterschied zu GoldFoundry
Dies ist die Version mit vollständiger Git-History. GoldFoundry (ohne -git) ist die aktive Working Copy. Bei Änderungen immer in BEIDEN prüfen welche aktueller ist.
