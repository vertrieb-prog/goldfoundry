```yaml
name: mlm-network

 Use when working on src/lib/mlm/, src/lib/points/, src/app/dashboard/partner/,
 or any affiliate/network marketing feature.

```

SKILL.md Inhalt:
```
# MLM Network System

## Architektur
- Closure Table (network_tree) für unbegrenzte Tiefe
- FORGE Points als interne Währung (1 FP = €0.10)
- 8 Einnahme-Quellen: Unilevel, Performance, Fast Start, 
  Matching, Compression, Builder Packs, Rang-Bonus, Pool
- 7 Ränge: Starter → Bronze → Silber → Gold → Platin → Diamond → Crown

## Kern-Regel
50% vom GESAMTEN Revenue geht an den Vertrieb.
- 50% von Subscriptions
- 50% vom Plattform-Anteil der Performance Fees (= 20% vom Profit)

## Compression
Inaktive User werden bei der Provision übersprungen.
Provisionen fließen zum nächsten aktiven Upline HOCH.
Berechnung: Filtere inaktive User aus der Depth-Berechnung.

## Vesting
Monat 1: 100% locked. Monat 2: 50% frei. Monat 3: 100% frei.
Bei Churn vor Monat 3: FP zurückbuchen.

## Builder Packs
5er (€99), 10er (€179), 25er (€399), 50er (€699)
Generiert Einladungs-Codes. Jeder Code = 1 User = 1 Email.
Pack-Kauf = sofortiger Rang-Boost.

## Tabellen
forge_points, fp_transactions, fp_achievements, fp_payouts,
network_tree, builder_packs, invite_codes, rank_history,
matching_bonus_log, performance_pool, pool_payouts

## Imports

import { TIER_RATES, RANK_REQUIREMENTS } from "@/lib/config";
```

## 📄 PARTNER LANDING PAGE SYSTEM

### Wie es funktioniert:
```
Route: /partner/[slug]/page.tsx (Dynamic Route)

Beispiel: goldfoundry.de/partner/max-mueller

1. Lade Partner-Profil aus profiles WHERE referral_slug = 'max-mueller'
2. Lade Partner-Stats: Netzwerk-Größe, Trader Performance
3. Lade ausgewählte Trader (Partner wählt im Dashboard)
4. Rendere personalisierte LP:
   - Partner Foto + Name + Bio
   - "Mein Netzwerk: 347 Trader · 72% Win Rate"
   - Trader-Karten mit Performance
   - CTA: "Jetzt starten — €6 im 1. Monat"
   - Link hat ?ref=max-mueller Parameter
5. Signup über diese Seite → automatisch dem Partner zugeordnet
6. Tracking: Visits, Signups, Conversions
```
